# Cellular Automata State Transitions

## Overview

This document contains state transition sequences for 4-cell cellular automata configurations using different rule combinations. Each configuration shows the evolution of states with binary representations and their decimal equivalents.

**Configuration Format:** `N Rule1 Rule2 Rule3 Rule4`  
**State Format:** `Binary State (Decimal Value)`
